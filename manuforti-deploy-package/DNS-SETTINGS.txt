CNAME RECORD (1 record)
-----------------------------------------------------------------

Replace [YOUR-GITHUB-USERNAME] with your actual GitHub username.

IMPORTANT NOTES
-----------------------------------------------------------------
1. These settings only work AFTER you enable GitHub Pages (Step 4)
2. DNS changes take 15-60 minutes to propagate
3. You can check if it's working at: https://manuforti.no
4. If you see a GitHub 404 page, wait longer for DNS

-----------------------------------------------------------------
TROUBLESHOOTING
-----------------------------------------------------------------
Problem: Site not loading
Solution: Wait 60 minutes, clear browser cache, try again

Problem: GitHub 404 error
Solution: Check that GitHub Pages is enabled in repository settings

Problem: DNS not working
Solution: Verify all 4 A records are added correctly
