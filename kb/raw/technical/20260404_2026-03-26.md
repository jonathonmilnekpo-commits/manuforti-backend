## Nemotron 3 Super Rate Limiting Issues

### Problem Identified
- March 26, 2026 (02:39-03:09 PDT): Multiple 429 errors from OpenRouter
- Error: "nvidia/nemotron-3-super-120b-a12b is temporarily rate-limited upstream"
- Affected heartbeat runs at 02:39, 02:54, and 03:09
- Root cause: OpenRouter shared tier infrastructure limits

### Impact
- Heartbeat failures during rate-limited windows
- No HEARTBEAT_OK responses sent during affected periods
- System retries failed repeatedly

### Potential Solutions Documented
1. Add direct Nvidia API key (bypass OpenRouter shared limits)
2. Reduce heartbeat frequency further (30 min instead of 15 min)
3. Add fallback model (Kimi if Nemotron fails)
4. Switch heartbeats to Kimi K2.5 (more reliable, no observed rate limits)
