## Lesson — Long Document Scripts (22:53)
- Always save (`doc.save(...)`) after each major change block in python-docx scripts
- Timeouts lose only the last unsaved step, not everything
- Pattern: version → save, section rewrites → save each one, insertions → save after each section
- This applies to any long-running file manipulation script
