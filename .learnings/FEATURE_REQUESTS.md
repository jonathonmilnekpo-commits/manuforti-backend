# Feature Requests

## Format

```markdown
## YYYY-MM-DD: Feature Name

**Priority:** critical / high / medium / low
**Effort:** small / medium / large
**Category:** tool / workflow / integration / capability

### Description
What the user wants

### Use Case
Why it's needed

### Proposed Solution
How to implement

### Status
- [ ] Pending
- [ ] In Progress
- [ ] Completed
- [ ] Declined
```
