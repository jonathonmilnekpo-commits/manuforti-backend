# Product 1 QC Framework: Nel ASA v15 as Gold Standard
**Version:** 1.0  
**Date:** 6 March 2026  
**Purpose:** Define comprehensive QC criteria for Product 1 reports, using Nel ASA v15 as the canonical reference

---

## 1. QC OVERVIEW

### 1.1 Philosophy
Nel ASA v15 represents the current state-of-the-art for Product 1. It should be stress-tested and improved upon, not treated as perfect. The goal is to:
- Identify gaps or weaknesses in Nel v15
- Establish what "excellent" looks like for each dimension
- Create actionable improvements for future reports

### 1.2 QC Dimensions (8 Total)

| Dimension | Weight | Description | Owner |
|-----------|--------|-------------|-------|
| Visual Quality | 20% | Design, charts, layout, branding consistency | Venture |
| Data Accuracy | 25% | Correctness of financials, metrics, facts | Researcher → Validator |
| Insight Depth | 20% | Actionable intelligence beyond surface data | Venture |
| Actionability | 15% | Clear recommendations, conditions, next steps | Venture |
| Template Compliance | 10% | Adherence to v15 structure and standards | Validator |
| Narrative Flow | 10% | Storytelling across slides, logical progression | Venture |

## 2. DIMENSION 1: VISUAL QUALITY (20%)

### 2.1 Chart Standards

| Element | Nel v15 Standard | QC Check | Pass/Fail |
|---------|------------------|----------|-----------|
| **Risk Gauge** | Semi-circular, 0-100 scale, needle at 48, 4 pillar boxes below | Gauge present? Correct score? Readable at glance? | |
| **Financial Chart** | Dual-axis: bars (revenue) + line (EBITDA), direct labels, insight box | Both axes present? Data labels clear? Insight explains "why"? | |
| **Market Position** | Horizontal bars, subject highlighted in navy, competitors in grey, annotation | Subject differentiated? Competitors named? Insight annotation? | |
| **Timeline** | Milestones with dates, alternating layout, connecting line | Chronological? Key events included? Visual clarity? | |
| **Risk Matrix** | 2x2 quadrants with zone labels, 6+ risks plotted, bubble size = severity | Quadrants labeled? Risks positioned logically? Bubble sizes meaningful? | |
| **Radar Chart** | 6 dimensions, subject vs benchmark, clear legend | All 6 dimensions? Benchmark appropriate? Scales 0-10? | |
| **Peer Risk** | Lollipop chart, risk zones shaded, subject highlighted | Zone bands visible? Subject stands out? Scores labeled? | |
| **ESG** | 3 columns (E/S/G), rating badges, controversy screening at bottom | All 3 pillars? RAG indicators? Controversies prominent? | |

### 2.3 Nel v15 Visual Stress Tests

**Strengths to Preserve:**
- Clean risk gauge with clear needle position
- Financial chart with insight box explaining negative EBITDA
- Market position chart with clear annotation
- ESG controversy screening prominently displayed at bottom

**Potential Improvements:**
- [ ] Timeline: Could include facility photos for major milestones
- [ ] Risk matrix: Bubble labels could be larger for readability
- [ ] Radar chart: Could add 2nd competitor for richer comparison
- [ ] Peer risk: Could add trend arrows (improving/deteriorating)

### 4.2 Insight Depth Checklist

- [ ] Every chart has a "so what" explanation
- [ ] Financial trends include "why" not just "what"
- [ ] Risks include business impact, not just description
- [ ] Market position includes competitive dynamics
- [ ] ESG includes materiality (what matters to buyer)

### 4.3 Nel v15 Insight Stress Tests

**Strengths:**
- Financial insight box explains negative EBITDA context well
- Risk matrix positions risks with business logic
- ESG clearly links to buyer's procurement decision

**Potential Improvements:**
- [ ] Executive Summary could include 1-line "what this means for buyers"
- [ ] Market position could include pricing dynamics (are they premium/discount?)
- [ ] Commercial slide could include negotiation leverage assessment
- [ ] Could include "red flags vs yellow flags" distinction in risks

## 5. DIMENSION 4: ACTIONABILITY (15%)

### 5.1 Actionability Standards

| Element | Nel v15 Standard | QC Check |
|---------|------------------|----------|
| **Recommendation** | Conditional Approval with rationale | Clear verdict? Conditions listed? |
| **Commercial Conditions** | 5 specific contract terms | Milestone-based, bonds, LDs, etc. |
| **Due Diligence** | 5 specific actions | Site visit, reference checks, etc. |
| **Risk Summary** | 4 pillars with ratings | All rated? Consistent with detail slides? |
| **Next Steps** | Implicit in due diligence list | Clear path forward? |

### 5.2 Actionability Checklist

- [ ] Buyer knows whether to proceed (Yes/No/Conditional)
- [ ] If conditional, specific conditions listed
- [ ] Contract terms suggested (payment, guarantees, etc.)
- [ ] Due diligence actions specified
- [ ] Risk mitigation strategies implied or stated
- [ ] Timeline for decision-making clear

### 5.3 Nel v15 Actionability Stress Tests

**Strengths:**
- Clear "Conditional Approval" verdict
- Specific commercial conditions (milestone-based, 10-15% bonds)
- Concrete due diligence list (site visit, reference checks)

**Potential Improvements:**
- [ ] Could include "if you proceed, negotiate for X" specific tactics
- [ ] Could include "if you don't proceed, alternatives are..."
- [ ] Could include timeline ("complete DD within 30 days")
- [ ] Could include "escalation triggers" (what would change the recommendation)

## 7. DIMENSION 6: NARRATIVE FLOW (10%)

### 7.1 Story Arc Check

| Transition | Logic | QC Pass? |
|------------|-------|----------|
| Slide 1 → 2 | Title sets context → Risk assessment delivers verdict | |
| Slide 2 → 3 | Risk overview → Specific recommendation | |
| Slide 3 → 4 | What to do → Who they are | |
| Slide 4 → 5 | Company profile → Financial health | |
| Slide 5 → 6 | Financials → Market position | |
| Slide 6 → 7 | Market context → Operational risks | |
| Slide 7 → 8 | Risks → Commercial terms | |
| Slide 8 → 9 | Commercial → ESG final check | |

### 7.2 Narrative Coherence

- [ ] Story builds logically from "who" → "how risky" → "what to do"
- [ ] No jarring jumps between topics
- [ ] Each slide answers questions raised by previous
- [ ] Final slide (ESG) reinforces or adds to recommendation

### 7.3 Nel v15 Narrative Stress Tests

**Strengths:**
- Clear progression: Risk overview → Deep dive → Action
- Financial slide explains context for recommendation
- ESG reinforces "green hydrogen enabler" positioning

**Potential Improvements:**
- [ ] Could include "narrative through-line" (e.g., "scale-up story")
- [ ] Could link market position more directly to operational risks
- [ ] Could connect financial losses to R&D investment more explicitly

## 9. NEL V15 AS GOLD STANDARD: ASSESSMENT

### 9.1 What Nel v15 Does Exceptionally Well

1. **ESG Controversy Screening**: Large, prominent red banner at bottom of Slide 9 — sets standard for visibility
2. **Financial Context**: Insight box explains negative EBITDA as "scale-up phase" — turns weakness into understandable context
3. **Risk Gauge Design**: Clean, readable, pillar summary boxes add value
4. **Recommendation Clarity**: "Conditional Approval" with specific conditions and DD list
5. **Market Position Annotation**: Clear callout explaining Nel's capacity roadmap

### 9.2 What Nel v15 Could Improve (Future Gold Standard)

1. **Data Completeness**: Add interest coverage, 3-year CAGR, gross debt figure
2. **Timeline Visual**: Include facility photos for Heroya/Wallingford
3. **Risk Matrix**: Larger labels, perhaps risk owner assignment
4. **Commercial Intelligence**: Add negotiation leverage assessment
5. **Executive Summary**: Include 1-line "what this means for buyers"

## 10. VENTURE QC WORKFLOW

### 10.1 Pre-Review Checklist

Before evaluating any Product 1 report:
- [ ] Open Nel v15 as reference
- [ ] Open target report side-by-side
- [ ] Have QC scoring worksheet ready
- [ ] Have this framework document open

### 10.2 Review Process

1. **First Pass (5 min)**: Flip through all 9 slides — does it feel professional?
2. **Second Pass (15 min)**: Evaluate each dimension using criteria above
3. **Third Pass (10 min)**: Compare specific elements to Nel v15
4. **Documentation (10 min)**: Fill scoring worksheet, note improvements
5. **Feedback (10 min)**: Write actionable feedback for Venture/Researcher

### 10.3 Output

For each QC review, produce:
- Completed scoring worksheet
- List of critical issues (if any)
- List of improvement opportunities (prioritized)
- Go/No-Go recommendation for delivery
