## Final Verification Checklist

### File Properties
- [ ] File size > 1,000 KB (indicates embedded charts)
- [ ] Exactly 9 slides
- [ ] All images embedded (not linked)
- [ ] No placeholder text

### Content Verification
- [ ] No "[Enter...]" or "[Placeholder]" text
- [ ] All financial numbers have units (€, $, %)
- [ ] Risk ratings consistent across slides
- [ ] Recommendation matches risk score
- [ ] No competitor names in wrong places

### Format Verification
- [ ] Navy header on all slides
- [ ] Source line on all slides
- [ ] Manu Forti logo on all slides
- [ ] Consistent font sizes
- [ ] Colors: Navy (#002147), Cobalt (#2B6CB0)

### Cross-Contamination Check
- [ ] No references to other suppliers (Nel, WEG, etc.)
- [ ] Correct company name throughout
- [ ] Correct industry context
- [ ] Correct stock exchange/ticker

## Common Failures to Avoid

| Failure | Cause | Prevention |
|---------|-------|------------|
| Missing charts | Sub-agent timeout | Generate charts directly, not via sub-agent |
| Text-only slides | No image embedding | Verify all 9 PNGs exist before PPTX build |
| Missing panels | Incomplete template | Use this checklist for every slide |
| Wrong file size | Charts not embedded | Check file size > 1MB before delivery |
| Cross-contamination | Copy-paste errors | Search for wrong company names before delivery |
